# installation


```
bash deps.sh
```
```
cp -r conf/* $HOME/.conf/ 
```

```
cp .bashrc $HOME
```

```
chmod +x startup.sh
```

```
cp startup.sh $HOME
```

# after boot


```
startx ./startup.sh 
```
